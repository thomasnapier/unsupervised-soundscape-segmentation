# Reproducibility materials for: An Efficient Pipeline for the Unsupervised Segmentation of Heterogeneous Natural Soundscapes

## Overview
This repository contains the code, processed data, and documentation required to reproduce the analyses, tables, and figures reported in the manuscript.

## Repository contents
- `data/processed/`: processed inputs used in the paper
- `src/`: source code for preprocessing, analysis, and plotting
- `scripts/`: ordered scripts to reproduce the workflow
- `results/`: generated figures and tables
- `docs/`: reproducibility notes and data dictionary

## Requirements
- Python 3.x
- See `requirements.txt` / `environment.yml`

## Reproducing the paper
1. Create the environment
2. Run `01_prepare_data.py`
3. Run `02_run_analysis.py`
4. Run `03_make_figures.py`
